DIY : 공작새 App Design

목록으로
  • 작업년도 : 2018
  • 작업범위 : 기획 · 사용자 조사 · 페르소나 설계 · 서비스 구조 설계 · UX/UI 디자인 · 아이콘 디자인
  • 사용툴 : Photoshop · Illustrator
  • 작업설명 :
    DIY 인테리어 정보를 제공하는 모바일 앱을 목표로, 사용자 조사부터 페르소나, UX 시나리오, 정보구조 설계, 와이어프레임, UI 디자인까지 전체 프로세스를 직접 진행한 프로젝트입니다. 타깃 사용자의 행동과 니즈를 분석해 서비스 방향을 정의했고, 인터페이스는 보라 계열의 컬러 시스템과 단순한 레이아웃으로 직관적인 사용 경험을 구성했습니다. 앱 아이콘 또한 스케치부터 컨셉 도출까지 직접 제작하여 브랜드 정체성을 완성했습니다.
