  • 사용언어 : HTML · CSS · JavaScript · jQuery

  • 작업설명 :
    Toss Web UI의 동적인 인터랙션을 중심으로 클론 제작한 프로젝트입니다.
    Lottie JSON(loan-ui.json)을 적용해 원본과 유사한 애니메이션을 구현했으며,
    jQuery 기반 메뉴 인터랙션과 AOS 스크롤 페이드 효과로 자연스러운 모션을 구성했습니다.
    또한 다양한 디바이스에서 동일한 사용자 경험을 제공하도록 반응형 웹으로 설계하고,
    모바일 메뉴 반응 및 clip-path 효과 등을 통해 토스 특유의 부드러운 UI를 재현했습니다.
