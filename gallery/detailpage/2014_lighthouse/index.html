Design Project : Lighthouse

목록으로
  • 작업년도 : 2014
  • 작업형태 : 2인 공동 프로젝트
  • 역할 : 리서치 / 아이콘 디자인 / 패턴 레이아웃 확장
  • 사용 툴 : Adobe Illustrator, Photoshop
  • 작업설명 :
    부산의 지역성을 시각적으로 표현하기 위해 ‘등대’를 모티브로 아이콘을 제작한 프로젝트입니다. 총 20개의 등대를 선정하여 형태적 특징을 단순화·재구성하였고, 이를 기반으로 패턴·워드마크·굿즈 시뮬레이션까지 확장 가능한 디자인 시스템으로 발전시켰습니다. 아이콘은 Illustrator로 제작되었으며, 컬러 구성과 적용 방식은 Photoshop을 활용해 시각적으로 정리했습니다.
